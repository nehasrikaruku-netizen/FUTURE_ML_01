## 📌 Project Overview

**FUTURE_ML_01** is a Machine Learning internship project focused on analyzing historical sales data and building a forecasting model to predict future sales trends.

This project demonstrates:

- End-to-end Machine Learning workflow
- Time-series forecasting implementation
- Feature engineering techniques
- Model evaluation using performance metrics
- Deployment using Streamlit dashboard

---

## 🎯 Project Objectives

- Clean and preprocess raw sales dataset
- Perform Exploratory Data Analysis (EDA)
- Extract meaningful time-based features
- Build forecasting models using Prophet & Regression
- Evaluate predictions using MAE and RMSE
- Create an interactive Streamlit dashboard

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository


git clone https://github.com/nehasrikaruku-netizen/FUTURE_ML_01.git

cd FUTURE_ML_01


### 2️⃣ Install Dependencies


pip install pandas numpy matplotlib seaborn scikit-learn prophet streamlit jupyter


Or create a `requirements.txt` file with:


pandas
numpy
matplotlib
seaborn
scikit-learn
prophet
streamlit
jupyter


Then install:


pip install -r requirements.txt

## 📑 Mini Project Report

### 1. Introduction

Sales forecasting is essential for business planning and demand management. This project focuses on predicting future sales using historical data and time-series techniques.

---

### 2. Dataset Description

The dataset contains historical sales data with:

- Date
- Sales amount

The data was cleaned and converted into a proper time-series format before modeling.

---

### 3. Methodology

#### Data Preprocessing
- Converted date column to datetime
- Sorted records chronologically
- Handled missing values

#### Exploratory Data Analysis
- Trend visualization
- Seasonality analysis
- Distribution checks

#### Feature Engineering
- Extracted year, month, quarter
- Created rolling averages
- Generated lag features

#### Model Development
- Implemented Prophet for forecasting
- Built regression-based models
- Compared performance metrics

- Achieved reliable prediction accuracy
